# SwiftUI_PictureInPicturePlaybackDemo


A demo of Picture-in-Picture playback that 
1. automatically start upon leaving the app
2. automatically stop when coming back to the app
3. looping 


For more details, please refer to my blog: [SwiftUI: Picture in Picture. Automatically start on Leaving the App. Automatically Dismiss when Coming back](https://medium.com/@itsuki.enjoy/swiftui-picture-in-picture-f16c496da4b7).
